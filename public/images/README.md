- `Figure_2_Analytical_Validation.png`
- `Figure_3_Hartmann_Study.png`

- `Figure_5_Prandtl_Study.png`

- `Figure_12_Literature_Validation.png`

## Recommended Format

- **Format:** PNG (preferred) or JPG
- **Resolution:** At least 1200px wide for clarity
- **Aspect Ratio:** 16:10 or 16:9 for best display

## After Adding Images

Update the `renderFigures` function in `src/App.js` to display your images.

Example:
```jsx
<img 
  src="/images/Figure_1_Grid_Convergence.png" 
  alt="Grid Convergence Study"
  style={{ width: '100%', height: '100%', objectFit: 'cover' }}
/>
```
